# Communication Modülü

`ace/communication` paketi sistem içerisindeki komut tanımlarını (`CommandId`), eksen seçimlerini (`AxisId`), dönüş yönlerini ve hata kodlarını listeler. `AxisManager` (uygulama çekirdeği) ile Protokol ayrıştırıcısı arasındaki C++ veri taşıyıcı modelleri (`CommandRequest`, `TelemetryResponse` vb.) burada tanımlanır.
