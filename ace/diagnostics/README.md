# Diagnostics Modülü

Sistemin donanım sağlığını (health points) test eder. Cihaz ilk tetiklendiğinde veya çalışırken düzenli aralıklarla voltaj, sıcaklık, limit tuşları ve enkoder bağlantı hataları gibi metrikleri izler. Sınır aşımı durumlarında sistemi uyararak Güvenlik katmanına (`safety`) devreder.
